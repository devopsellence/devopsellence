## Local

The scaffold does not require Rails or mise to generate. For local Rails work:

```sh
mise install
mise exec -- bundle install
mise exec -- bin/rails db:prepare
mise exec -- bin/rails server
```

If you already have Ruby and Bundler available, the same commands work without
`mise exec --`.

## Check

```sh
scripts/check
```

## Deploy

Local deploy workflows still build the Dockerfile on this machine.

```sh
devopsellence deploy --dry-run
```
